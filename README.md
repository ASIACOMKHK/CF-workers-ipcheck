IP Checker - Cloudflare Workers
一个轻量、快速的 IPv4 / IPv6 地址检测与地区查询工具，基于 Cloudflare Workers 部署，全球低延迟访问。
✨ 功能
获取访问者真实 IP（IPv4 / IPv6）
自动返回 IP 归属地区信息
响应速度快，无服务器依赖
可直接部署到 Cloudflare Worker
🚀 快速部署
1.克隆仓库
git clone https://github.com/你的用户名/仓库名.git
cd 仓库名
2.安装依赖
npm install
3.本地调试
npm run dev
4.一键部署到 Cloudflare Workers
npm run deploy
📌 使用示例
访问你的 Workers 域名即可直接获取 IP 信息
https://your-worker.your-subdomain.workers.dev
